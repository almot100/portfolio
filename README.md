# Portfolio – Moussa Terap Ali

Portfolio personnel présentant mes projets en génie informatique, 
cybersécurité et systèmes embarqués.

## Aperçu
Site web statique développé en HTML, CSS (Tailwind) et JavaScript.
Il met en valeur mes projets académiques et personnels, 
ainsi que mes compétences techniques.

## Technologies utilisées
- HTML5
- Tailwind CSS
- JavaScript
- Vanta.js / Three.js
- Typed.js

## Sections
- Accueil
- À propos
- Projets
- Contact

## Objectif
Support de candidature pour stages et opportunités professionnelles 
en génie informatique (cybersécurité / systèmes embarqués).
